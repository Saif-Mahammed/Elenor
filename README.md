# ELENOR OMNI

The future of autonomous, visually intelligent, and emotionally aware personal AI.

## Features
- **Omni Intelligence:** Hybrid cloud/local brain for zero-cost scalability.
- **Visual Intelligence:** Real-time screen and camera perception using local models.
- **Proactive Memory:** Long-term relationship tracking and personalization.
- **Plugin Marketplace:** Extendable architecture for 40+ features.
- **Hardware Master:** Full control over system volume, brightness, battery, and media.

## Quick Start
1. `conda activate jarvis`
2. `pip install -r Requirements.txt`
3. `python Main.py`
